# BTC Streamlit App 📊

Cette application affiche un **graphique en chandeliers du BTC/USD** avec plusieurs timeframes :
✔️ **5 minutes**  
✔️ **15 minutes**  
✔️ **1 heure**  

Les données sont récupérées en temps réel via l'API Binance.

## 🔧 Technologies utilisées
- **Python**
- **Streamlit**
- **Plotly**
- **Pandas**
- **Requests**
